# `finchers-todo`

[finchers] で ToDo アプリを作るサンプル

[finchers]: https://github.com/finchers-rs/finchers

## Usage

```command
$ cargo run -- --host 127.0.0.1 --port 7878
```

```command
$ curl http://127.0.0.1/
Hello, Heroku!
```
